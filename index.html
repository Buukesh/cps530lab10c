This website is powered by Vue.js, a JavaScript framework.

Setting up this Website

I set up this Vue.js website, following these steps:

  1. Create an HTML file and include the necessary structure.
  2. Create a CSS file for styling and link it in the HTML file.
  3. Include the Vue.js library in the HTML file using a CDN invocation.
  4. Create a JavaScript file for the Vue.js application logic.
  5. Write my Vue.js code inside the setup function.
  6. Mount the Vue app to the specified element (in my case, #app).

Difficulties with Vue.js

One of the difficulties I faced was how to correctly set up a Vue.js project with the proper file structure, especially when it came to integrating the app.js script which housed the Vue app. Intially, I was unsure what the logic would be more suited for the html file or the app.js file. However, reading the documentation proved to be a valuable resource in clarifying this confusion. The html file is more concerned with structural layout and static content. On the other hand, the app.js logic deals with dynamic content and updates to the DOM. I was able to gather this from the high-level introduction in the docs.
